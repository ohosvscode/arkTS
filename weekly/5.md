# 实习周记 第五周（2月9日 - 2月13日）

本周我继续在 ArkTS 插件中继续推进 webview 相关的功能与依赖更新：完成 Device Manager 面板的落地、接入 unplugin-vue-i18n 做编译时国际化，并将 @arkts/sdk-downloader 升级至新版本并解决其修复的若干问题。

Device Manager 面板：在统一前端架构下新增设备管理面板，包含设备列表、创建设备、镜像管理等页面，与 HDC Manager 共用 HDC 连接与协议，用户可在同一套 webview 中管理设备与镜像。通过 device-manager-command 注册相关命令，并在 frontend 的 pages/device-manager 下实现列表、创建与镜像管理界面。

国际化：接入 unplugin-vue-i18n，在 Vite 编译阶段加载 i18n 文案，与插件的 package.nls.*.json 对齐，便于后续多语言维护与扩展。

@arkts/sdk-downloader 升级与修复：将 @arkts/sdk-downloader 升级至 v0.1.32。该版本修复了 Windows 下的若干问题，将 API20-Beta1 更新为 API20-Release，并修复了 native 工具链的 symlink 问题（对应 sdk-downloader 仓库的 issue#17），使在 Windows 与多环境下使用 SDK 下载与 native 工具链时更稳定。插件在依赖升级后进行了验证，确保 SDK 下载与相关功能正常。
