# 实习周记 第三周（1月27日 - 1月31日）

本周我继续在 ArkTS 插件项目上推进开发，主要围绕前端架构重构与构建链路调整展开（对应后续合并的 PR#262 / feat: #48 的前半部分工作）。

此前插件的 webview 由多个独立的 SPA 项目组成（如 qualifier-editor、project 等），每个单独打包，无法跨项目做 tree-shaking，体积偏大。本周我将这些前端合并为一个统一的 SSG（静态站点生成）项目，放在 packages/vscode/src/frontend/ 下，使用 vite-ssg 配合 Vite 构建。这样可以在一次构建中生成多个静态页面，并利用 Vite 的 tree-shaking 去掉未引用代码。

具体包括：在 vite.config.mts 中配置 SSG 选项（如异步脚本、样式收集与注入、NLS 缓存清理等），并接入 @css-render/vue3-ssr 处理 SSR 样式；统一前端目录结构，按 pages/、components/、composables/、functions/、interfaces/ 等划分，各面板共用同一套路由、布局和构建入口；重构与扩展端的通信方式，为每个面板维护各自的 connection protocol 和 server function，与 unioc/vscode 的 Translator 等集成，便于后续新增面板。

重构后，在面板数量增加的前提下，webview 总体积由原来的约 4MB 继续降至约 3MB，为下一周新增 HDC Manager、Device Manager 等新的 WebView 面板打好基础。
