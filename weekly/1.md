# 实习周记 第一周（1月8日 - 1月9日）

本周是我实习的第一周，我将继续围绕着开源ArkTS插件（https://github.com/ohosvscode/arkTS）项目做主要维护工作。

我合并并落地了 PR#246，主要做了三件事：一是移动语言配置，将 language-configuration.json 调整到更合适的位置，便于后续维护；二是移除 HML 语法支持，删除了 syntaxes 目录下的 hml.tmLanguage.json 和 hml-language-configuration.json；三是修复 Lint 问题，更新了 eslint.config.js、pnpm-workspace.yaml 以及 packages/vscode/package.json 等相关配置，保证代码规范与构建通过。

删除 HML 语法主要有两点考虑：第一，HML 已被华为官方标记为废弃（deprecated），不再作为推荐技术栈；第二，当前插件在 syntaxes 目录下若同时保留两个 tmLanguage 语法文件，会与我在 github-linguist/linguist 仓库中提交的 PR#7742 产生冲突（linguist 对语法文件的识别与统计会受影响）。因此我计划先在本插件中移除 HML 支持，以保持语法定义清晰，并配合上游 linguist 的 PR 推进。
